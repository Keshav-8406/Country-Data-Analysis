# 🌍 Country Data Analysis using REST API

---

## 📌 Project Overview
This project demonstrates how to fetch real-time country data from a public REST API and analyze it using Python.

It simulates a real-world data analyst task where data is retrieved programmatically, cleaned, and summarized for insights.

---

## 🚀 Technologies Used
- 🐍 Python  
- 📦 Pandas  
- 🌐 Requests (API handling)  
- 📊 Jupyter Notebook (Google Colab)

---

## 🔗 API Used
- REST Countries API  
👉 https://restcountries.com/v3.1/all  

---

## 📂 Tasks Performed

### ✅ Task 1 — Fetch & Validate Data
- Sent GET request using `requests`
- Handled errors using `try-except`
- Validated response (Status Code: 200)
- Retrieved total number of countries

---

### ✅ Task 2 — Data Extraction & Cleaning
- Extracted:
  - 🌍 Country Name  
  - 👥 Population  
  - 🗺️ Region  
  - 📐 Area  
- Converted JSON → Pandas DataFrame  
- Displayed first 10 records  

---

### ✅ Task 3 — Data Analysis
- 📊 Identified region with highest population → **Asia**  
- 🌏 Found country with largest area → **Russia**

---
